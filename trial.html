Step 1: Simple Integration (Recommended)

Add this single line to your HTML file (before closing </body> tag):

<script src="https://query-bee-mauve.vercel.app/widget-standalone.js"></script>

That's it! The widget will automatically appear on your website.

Step 2: Alternative Method (Full Control)

If you want more control, add both styles and script:

<!-- QueryBee Widget Styles --> <link rel="stylesheet" href="https://query-bee-mauve.vercel.app/widget-styles.css"> <!-- QueryBee Widget Script --> <script src="https://query-bee-mauve.vercel.app/widget-standalone.js"></script>

Step 7: Troubleshooting

If widget doesn't appear:

  • Check browser console for errors
  • Ensure script URL is correct and accessible
  • Verify no JavaScript conflicts on your page
  • Try clearing browser cache

If chat doesn't work:

  • Check if backend API is running
  • Verify network connectivity
  • Check browser console for API errors
